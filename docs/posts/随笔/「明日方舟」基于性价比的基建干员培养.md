## 前言

我是一名轻度游戏的开服玩家，主要游玩日常以及看主线剧情。从19年到现在总共玩了3个号，中间 AFK 过两次。上一次 AFK 是2022年夏活初，原因是对二游的打卡式的日活感到烦躁，特别是之前没有 MAA 以及一键换班的时候。现在的号是在 2024-07-31 创建的，目前是115级。

在之前的版本中，我都是抄 B 站 UP 主[公孙长乐](https://space.bilibili.com/22606843)的作业，太过于追求高收益。在今天看了[《基建强度也是强度！兔头党如何拥有高水平基建》](https://www.bilibili.com/video/BV1LoHwzBE8e?spm_id_from=333.788.videopod.sections&vd_source=82db1cc095658bd54b0f2bb90b33953e)后有了不一样的想法。基建是明日方舟获取资源的重要方式，合理的基建排班对于资源有切实的提升。本文会以 243（自动轮换）基建布局为标准，记录下有性价比的养成方式。

## 为什么是 243 自动轮换？

基建布局分为153、252、243、333，其中243布局兼顾了赤金、贸易与作战记录，并且可以在不拆设施的基础上自由切换挫玉。243 分为两队列与三队列，其中两队列的模式虽然收益不及三队列，但支持自动轮班，哪里红了点哪里。“人间烟火”、“感知信息”这样的体系不在我的考虑范围内，一来培养成本极高，二来需要对齐排班（对于游玩一款二游，心智负担太大）。

## 如何算是有性价比

以制造站生产作战记录的红松林骑士团为例，需要精2的干员有：薇薇安娜、焰尾、远牙、野鬃、灰毫，只为了基建去培养这样一堆中杯干员实在令人望而却步。性价比的基建应该首当考虑能否在游玩中使用到干员，其次考虑获取难度与培养成本。单纯为基建去培养干员，同加成下的低星优于高星。

## 推荐组合

> - 243 中常态为 2贸易 + 2赤金 + 2作战记录，短期挫玉可切换为 1贸易订单 + 1源石订单+ 2赤金 + 1作战 + 1源石制造。
> - 制造中4个主队列效率 100% 以上，4个副队列不要在 90% 即可。
> - 没有明确说明练度的干员可为任意练度。
> - 以下推荐的组合效率由高到低排序。

### 贸易站

- 龙舌兰组：精2龙舌兰 + 精2巫恋 + 精2裁缝干员（柏喙/明椒/卡夫卡/折光，任选其一）
- 但书组：精2(但书/可露希尔) + 精2深巡 + 精2吉星
  - 伺夜、空弦、赫德雷、空弦这类加 40% 效率干员可替换
- 拉特兰：精2蕾缪安 + 精2新约能天使 + 精2空弦
  - 旧能和蕾缪安 > 新能和蕾缪安，但是旧能目前没有上场空间，已经培养过的可以上
- 格拉斯哥帮：推王 + 精2异格推王 + 精2摩根
  - 推王无需培养可公招，摩根赠送

龙舌兰组和但书组仅作用于赤金订单，建议放到同一贸易站。拉特兰和格拉斯哥帮可以挫玉，建议放到同一贸易站。

### 赤金制造站

- 精1清流 + 精2温蒂 + 精2森蚺（精1冬时可平替）+ 精2承曦格雷伊（放于发电站）
  - 这套缺点在于刚需温蒂，不玩生息演算直接用冬时平替森蚺
- 精2槐琥 + 精2阿罗玛 + 精1砾/精2引星棘刺
- 精2苍苔 + 精1斑点 + 夜烟
  - 斑点、夜烟可换任意“金属工艺”技能干员，练了引星棘刺可替换以提高产能

### 作战记录制造站

- 精2酒神 + 精2克里斯汀 + 精1红云
- 精2至简 + 精2弑君者 + 精1断罪者/精2食铁兽
- 水月组：精2水月 + 两个任意标准化β技能
  - 整体相当于 90% 效率，优点在于水月有特限模组
- 火神泡泡：精2火神 + 精1泡泡 + 贝娜
  - 整体效率不够看，优点在于火神可在蚀刻章列表中领取兑换，并且切换挫玉时不易爆仓。

### 控制中枢

- （精2斩业星熊 + 诗怀雅） + （精2维什戴尔 + 魔王） + 精2八幡海铃（任一笑脸或体系拐）
  -  八幡海铃作为办公室效率加成等于黄票加成等于抽卡资源，收益非常高，243情况下甚至高于直接的贸易站和制造站效率，有则必练。
  - 这里没有使用阿米娅，因为阿米娅只能减魔王一个人的心情消耗。除非里设有用诗怀雅+斩业星熊，而是凯尔希+烛煌。
- 精2阿斯卡纶 + 精2Mon3tr + 精2玛恩纳 + 任意笑脸干员（用于叠玛恩纳被动）
